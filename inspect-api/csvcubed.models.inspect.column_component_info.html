csvcubed.models.inspect.column_component_info¶

ColumnComponentInfo¶

Contains the ColumnComponentInfo class which relates CSV Columns back to their qube-config.json style column types and the underlying RDF Data Cube DataStructureDefinition Components.

class csvcubed.models.inspect.column_component_info.ColumnComponentInfo(column_definition: ColumnDefinition, column_type: EndUserColumnType, component: QubeComponentResult | None)¶

Bases: object

Relates CSV Columns back to their qube-config.json style column types and the underlying RDF Data Cube

DataStructureDefinition Components.

column_definition: ColumnDefinition¶
column_type: EndUserColumnType¶
component: QubeComponentResult | None¶
The component may be None in situations such as:

  • The column is marked as suppressed,

  • The column is an observations column in a standard shape cube.
