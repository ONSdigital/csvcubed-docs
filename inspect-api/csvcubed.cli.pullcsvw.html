csvcubed.cli.pullcsvw.pull module¶

Pull CSV-W¶

Functionality to pull a CSV-W and all of its relative dependencies to the local filesystem.

exception csvcubed.cli.pullcsvw.pull.AbsoluteBasePathError(base_url: str)¶

Bases: ValueError

An exception to communicate that the base path of the CSV-W metadata document is an absolute URL.

This means that there are no relative dependencies that we can download. This exception should be caught and a warning communicated to the user.
