csvcubed.utils.uri¶

URI¶

Functions to help when working with URIs.

csvcubed.utils.uri.csvw_column_name_safe(label: str) → str¶

Converts a generic string into a string which is safe as the name property in a CSV-W column.

Returns:

A str based on label which is safe to use to name columns in a CSV-W metadata file.

csvcubed.utils.uri.ensure_looks_like_uri(value: str) → None¶

Ensure that value looks like a URI.

Raises:

ValueError – when value does not look like a URI.

csvcubed.utils.uri.ensure_values_in_lists_looks_like_uris(values: list[str]) → None¶

Ensure that the values in a list (values) look like URIs.

Raises:

ValueError – when value does not look like a URI.

csvcubed.utils.uri.file_uri_to_path(file_uri: str) → Path¶

Returns a normalised file path from a URI. Will work on both Windows and Linux/Unix.

csvcubed.utils.uri.get_data_type_uri_from_str(data_type: str) → str¶

Is it a uri? Find out, and return one.

csvcubed.utils.uri.get_last_uri_part(uri: str) → str¶

Gets the last segment of a URI’s path.

Returns:

The segment of the URI after the last /.

csvcubed.utils.uri.looks_like_uri(maybe_uri: str) → bool¶

Tests whether a str looks like a URI.

Returns:

whether the str looks like a URI or not.

csvcubed.utils.uri.uri_safe(label: str) → str¶

Convert a label into something that can be used in a URI path segment.

The function formerly known as pathify().
